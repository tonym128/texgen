# 3D Maze Explorer Example

A first-person 3D maze game powered by Three.js and real-time procedural textures from `texgen.js`.

## Features

- **Real-time Animated Textures**: Walls, floors, and ceilings are generated and animated on the GPU every frame.
- **Varied Wall Types**: Procedural Fire, Plasma, Cyber, Toxic, and Ice textures.
- **Fog of War**: A dynamic 2D minimap that reveals the maze layout as you explore.
- **Infinite Scaling**: Finding the exit portal (glowing sphere) generates a larger, more complex maze.
- **Smooth Navigation**: Slide-based collision detection for easy first-person exploration.

## Controls

- **WASD / Arrow Keys**: Move and Turn.
- **Mouse**: Not used (keyboard only).
- **Minimap**: Look at the bottom right to track your explored route.
