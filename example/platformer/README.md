# Ultimate Platformer Example

A 2D action platformer where all visual assets (player, enemies, blocks, backgrounds) are generated procedurally at runtime using `texgen.js`.

## Features

- **Procedural Themes**: Switches between Nature, Industrial, Castle, Cyber, and Ocean themes.
- **Dynamic Physics**: Smooth bounding-box collision detection with squash and stretch animations.
- **Persistence**: 5 save slots using `localStorage`.
- **Advanced Level Generation**: Scaling difficulty with chasms and mid-air platforms.
- **Cinematic Transitions**: Side-swipe transitions for deaths and level starts.
- **Drone Show**: A celebratory choreographed drone formation at the end of every stage.

## Controls

- **WASD / Arrow Keys**: Move and Jump.
- **Space / Enter**: Jump (in-game) or Start Level (on world map).
- **Mouse**: Navigate UI and World Map.
