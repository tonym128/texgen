# Marble Roller Example

An orthographic 2.5D physics arcade game featuring procedural marble and environmental textures.

## Features

- **2.5D Physics**: Acceleration, friction, and wall-bounce mechanics.
- **Procedural Textures**:
    - **Marble**: Dynamic swirly patterns with specular highlights.
    - **Techno-Floor**: Hexagonal grid generated using Voronoi noise.
    - **Walls**: Wood-grained vertical blocks.
- **Smart Spawning**: Guaranteed fair starts with safety buffers and opposite-corner goal placement.
- **Infinite Levels**: New random layouts generated upon reaching the goal.

## Controls

- **WASD / Arrow Keys**: Roll the marble.
- **Goal**: Reach the pulsing cyan ring to progress and increase your score.
- **Hazard**: Avoid the black holes!
